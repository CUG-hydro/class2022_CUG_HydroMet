## 2. 如何把课件从md编译成pdf

md文件共分为两种，他们的使用方法分别是：

- 带`_doc`后缀：编译成**A4 pdf**格式（复习使用，`Typora`导出pdf）。

- 无`_doc`后缀：编译成**PPT格式的pdf**（课堂课件，`VSCode`+`Marp`导出pdf）。
